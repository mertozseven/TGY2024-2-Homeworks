
# iTicket

Bus ticket app made with UIKIT Programmatically.


## Features

- UIKit Programmatically
- QR Code Generator
- Light/dark mode
- No 3rd party packages

## Nice Have

- CoreData or Firebase integration
- Passkit integration
- Real Bus Data
